*መደምደምታ*


ኣህጉር ኣፍሪቃ ብዝተወደበ ሕጋዊ ስርዓትን ትካላትን ብምስራሕ፡ ንሰላም፡ ጽድቅን ምዕባለን ይቃለስ። ሕብረት ኣፍሪቃን ካልኦት ኣህጉራዊ ትካላትን ኣብ ምውሓስ መሰላት ዜጋታት ኣፍሪቃን ምዕባይ ቁጠባን ወሳኒ ግደ ኣለወን።



 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

         ወተሃደራዊ ኣሰራርሓ 

ኣገላልፃ፥ 

ወተሃደራዊ ኣገባብ፡ ንኽልቲኡ ሓፈሻዊ ገበናዊ ሕግን ፍሉይ ሕግታት ዕጡቕ ሓይልታትን ብምኽባር፡ ብወተሃደራዊ ኣባላት ወይ ኣብ ወተሃደራዊ ኩነታት ንዝፍጸሙ ገበናት ኣተሓሕዛ ዝምልከት እዩ። 

 

ቀንዲ ባህርያት፤ 

 

ጽኑዕ ስርዓተ-መሰረት (Strict Hierarchy)፦ ውሳነታት ኣብ ሰንሰለት ትእዛዝ ዝጣበቑ እዮም። 

 

ወተሃደራዊ ስነ-ስርዓት፡- ፍሉያት ገበናት ከም ምውጻእ፡ ዘይምእዛዝ፡ ወይ ስልጣን ምግዳፍ ዝኣመሰሉ። 

 

 

ወተሃደራዊ ቤት ፍርዲ፡ ብመሰረት ሕጊ ወተሃደራዊ ፍትሒ ንወተሃደራዊ ኣባላት ክፈርድ ብቑዕ። 

 

ካብ ሲቪላዊ ገበናዊ መስርሕ ዘሎ ፍልልይ፤ 

 

እቲ ኣገባብ መብዛሕትኡ ግዜ ዝቐልጠፈን ጽኑዕ ናይ ግዜ ገደብ ዝግበረሉን እዩ። 

 

ዝተወሰኑ መከላኸሊታት (ንኣብነት፡ ሕጋዊ ምኽሪ) ምስ ወተሃደራዊ ኩነታት ክመዓራረዩ ይኽእሉ። 

 

 

እገዳታት ስጉምትታት ዲሲፕሊን (መጠንቀቕታ፣ ምውራድ፣ ምስጓግ) ከጠቓልል ይኽእል። 

 

 

 

 

 

 

 

 

ተዋሳእቲ፤ 

 

ኣዛዚ ወይ መራሒ ኣሃዱ፡- መስርሕ ዲሲፕሊን ከበግስ ይኽእል። 

 

ወተሃደራዊ ዓቃቢ ሕጊ፡ ወተሃደራዊ ገበናት ምኽሳስ የረጋግጽ። 

 

 

ወተሃደራዊ ዳኛ፡- ንጉዳያት ብመሰረት ፍሉይ መለክዒታት ይውስን። መትከላት፤ 

 

ውሕስነት መሰል ምክልኻል። 

 

 

ቅልጣፈን ብቕዓትን ናይቲ ፈተነ። 

 

ወተሃደራዊ ሕግን ሕጊ ቅጽዓትን ብጥብቂ ምትግባር። 

 

 

 

 

 

                 

            *ስነ-ገበን* 

 

1. ገበን ምርዳእ ማለት ሕብረተሰብ ምርዳእ ማለት እዩ። 

 

2. ገበን ተግባር ጥራሕ ኣይኮነን፤ ንፍትሒ ብድሆ እዩ። 

 

3. ስነ-ገበን ኣብ ዘይርአ ጠንቅታት ገበን ብርሃን ይህብ። 

 

4. መቕጻዕቲ ሕነ ምፍዳይ ዘይኮነስ፡ መሳርሒ ፍትሕን ምሕዋይን እዩ። 

 

5. ማሕበራዊ ምቁጽጻር ዋላ እቲ ሕጊ እኹል ኣብ ዘይኮነሉ እዋን ስርዓት ይሕሉ። 

 

6. ገበናዊ ፍትሒ ልቢ ሕጊ ኣንጻር በደል እዩ። 

 

7. ገበን ትሕቲ ዕድመ ትምህርታውን ማሕበራውን ጉድለታትና ይገልጽ። 

 

8. ዝተወደበ ገበን ንናጽነትን ቁጠባን ሓንቲ ሃገር ኣብ ሓደጋ ዘእቱ እዩ። 

 

9. ገበነኛ ምሕዋይ ሕብረተሰብ የሐይል። 

 

10. ኩሉ መርመራ ሓቅን ፍትሕን ምድላይ እዩ። 

 

 

 

 

ቅዋም፡- ህዝባዊ ስልጣን ዝውድብን መሰል ዜጋታት ዘውሕስን መሰረታዊ ሕጊ። 

 

1. ምፍልላይ ስልጣን፡- ንሓጋጊ ኣካል፡ ፈጻሚ ኣካልን ፍርዳዊን ጨናፍር ዝፈሊ መትከል። 

 

2. ልዕልና ሕጊ፡- መንግስቲ ሓዊሱ ኩሉ ሰብ ኣብ ትሕቲ ሕጊ ዝግዛእሉ መትከል። 

 

3. መሰረታዊ መሰላት፡ ንነፍሲ ​​ወከፍ ውልቀ ሰብ ኣፍልጦ ዝወሃቦም ናጽነታትን ውሕስነትን (ናጽነት፡ ማዕርነት፡ ክብሪ)። 




 

4. ልኡላውነት፡- መንግስቲ ኣብ ልዕሊ መሬቱን ህዝባን ዘለዎ ላዕለዋይ ስልጣን። 

 

5. ቅዋማዊ ዳግመ ግምት፡ ሕግታት ምስ ሕገ መንግስቲ ዝሰማምዑ ምዃኖም ንምርግጋጽ ዝተዳለወ ኣገባብ እዩ። 

 ስለዚ ክብርን ልዕልና ሕብረተሰብ ሃገርን ብልእልና ሕዚጊ እዩ ዝምዘን
